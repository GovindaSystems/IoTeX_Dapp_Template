Bem-vindo ao repositório do IoTeX Dapp Template, um projeto que visa fornecer uma experiência de usuário completa para o nosso marketplace descentralizado. Este projeto é construído sobre a rede IoTeX, uma alternativa de baixo custo à Ethereum, comumente usada para automação IoT com protocolos MQTT.

## Conteúdo

Este projeto inclui:

- **Marketplace**: Cadastro de produtos, fornecedores, taxas de pagamento e um fluxo de checkout fácil.
- **Checkout**: Um processo de checkout simples e intuitivo.
- **Gateway de Pagamento**: Uma solução de pagamento integrada.
- **Carrinho de Compras**: Um carrinho de compras para gerenciar suas compras.
- **Área Logada**: Uma área segura para usuários logados.
- **Perfil de Usuário**: Um perfil de usuário personalizado.

## Como Baixar

Para baixar o projeto, você pode clonar o repositório usando o seguinte comando git:

```bash
git clone https://github.com/GovindaSystems/IoTeX_Dapp_Template.git
```

## Como Instalar 

Depois de clonar o repositório, você pode instalar as dependências necessárias usando o npm ou yarn. Navegue até a pasta do projeto e execute:

```bash
npm install
```
ou
```bash
yarn install
```

## Como Executar

Para executar o projeto localmente, você pode usar o comando:

```bash
npm run start
```
ou
```bash
yarn start
```

## Como Fazer o Deploy

Para fazer o deploy do projeto, você pode usar o comando:

```bash
npm run build
```
ou
```bash
yarn build
```

Depois de criar a build, você pode fazer o deploy no servidor de sua escolha.

## Como Cadastrar um Produto

Para cadastrar um produto, você deve acessar a área de administração do site e seguir as instruções para adicionar um novo produto.

## Como Cadastrar um Gateway de Pagamento

Para cadastrar um gateway de pagamento, você deve acessar a área de administração do site e seguir as instruções para adicionar um novo gateway de pagamento.

## Como Configurar o Gateway de Pagamento

A configuração do gateway de pagamento pode variar dependendo do provedor de gateway de pagamento que você está usando. Você deve seguir as instruções fornecidas pelo seu provedor de gateway de pagamento.

## Como Gerenciar Cashbacks

Para gerenciar cashbacks, você deve acessar a área de administração do site e seguir as instruções para gerenciar cashbacks. Você pode definir a porcentagem de cashback para cada produto e monitorar o cashback dado aos usuários.
